geoworkflow.utils.file_utils#

File and directory utilities for the geoworkflow package.

This module provides functions for file operations, temporary directory management, and file system utilities.

geoworkflow.utils.file_utils.buffer_geometry(geometry_gdf, buffer_distance_km)[source]#

Buffer geometries by a specified distance in kilometers.

For WGS84 coordinates, this uses a simple degree-based approximation. 1 degree ≈ 111 km at the equator (less accurate at poles).

Parameters:

  • geometry_gdf (GeoDataFrame) – GeoDataFrame with geometries to buffer

  • buffer_distance_km (float) – Buffer distance in kilometers

Return type:

GeoDataFrame

Returns:

GeoDataFrame with buffered geometries

Raises:

GeoWorkflowError – If buffering fails
